题目一 优化

使用jQuery和lodash优化Homework 5~7中的Games，包括Calculator，Maze, Whac-a-mole, Fifteen Puzzle。jQuery和lodash使用如下版本：

https://cdnjs.cloudflare.com/ajax/libs/jquery/2.1.0/jquery.js 

https://cdnjs.cloudflare.com/ajax/libs/lodash.js/3.10.1/lodash.js

提交包括优化前代码和优化后代码，并在README中说明

  1. 优化前后的LoC（代码行数）
  2. 自己Toolkits的使用心得

帮助Reviewer评判。

 

需求规格（50分）

 

  1. Games功能如旧，完全正常（30分，一个Game不正常扣8分，扣完为止）
  2. 代码减少（10分，一个Game做不到扣3分，扣完为止）
  3. 代码可读性增强（10分，一个Game做不到扣3分，扣完为止）

 

软件质量要求（10分）

 

JavaScript代码模块化，命名合理（5分），每个方法不超过10行（5分）

题目二 Table Sorter 

在给定源代码：source-code.zip的基础上，完成Table sorter。

Table sorter包括JavaScript和一点CSS，能够让原始的html table变得可以分别按照各栏数据值，对各行排序。

效果

  1. 在表头任意一个栏目中点击一下，下面各行将按照此栏目值的升序排序
    1. 按照字符串比较来确定顺序
  2. 再次点击该栏目，变更为降序排序
  3. 点击其它栏目，则按其它栏目的值重新排序
  4. 注意，排序时，栏目奇偶行的背景色保持奇数白色、偶数浅灰色
未点击点击再击点击其它

* 虽然上面例图中只显示了To-Do表格的变化，实际上Staff表格也是sortable的。

要求

  1. 使用jQuery，lodash等，让代码尽量简洁
  2. 尝试使其它网页的table变得sortable。具体做法是打开一个有table的网页，开启控制台，然后在其中输入一行你的神秘代码 ，网页上所有的tables变得可以排序。在提交文件中加上一个README文件，列举出可sortable的网页，和你具体的神秘代码

需求规格（30分）

  1. UI、交互和上图完全一致（10分）
  2. 能够排序(10分）
  3. 其它网页（10分，一个网页3分，封顶10分）

软件质量要求（10分）

JavaScript代码模块化，命名合理（5分），每个方法不超过10行（5分）

      一句话，$('...')里面的...是css的selector，语句将返回所有符合selector的元素。你这里的selector大概应该这样写吧$(“th[name=" + event.target.name + "]”) 。

      这里你的问题应该是在事件处理器里面如何访问父/祖先元素吧，利用jQueyr的parents、closest方法即可。

      其实，这个sorter很简单，理解Event的bubbling，用好jQuery的delegate，一切易如反掌。

      课程里要求大家下来先仔细看一遍jQuery和lodash的文档，再做作业，看来大家并没有。
